# Домашнее задание 2.3 — Aiohttp

REST API объявлений на aiohttp (аналог задания 2.1 Flask).

- POST /advertisement — создание
- GET /advertisement — список
- GET /advertisement/{id} — по id
- PATCH /advertisement/{id} — обновление
- DELETE /advertisement/{id} — удаление

Запуск: `pip install -r requirements.txt && python app.py`

Проверка: запустить контейнер (см. задание 2*) и проверить роуты.
